What is Lin?

‚ñæ


The Lin Method, created by Tse-Kan Lin (not Brandon Lin), is a speedsolving method for the Square-1 that is similar to Roux for the 3x3.

Its steps are:

1. Cubeshape
2. First Block
3. Second Block + DB edge
4. Corner Permutation + solve DF edge
5. Permute Last Layer Edges

Note: There are lots of variations within and throughout these steps, especially as you get more advanced. The ones I listed are what a beginner would need to learn.

I learned Lin using this tutorial by BenChrzti on Youtube, but there are many others out there.
